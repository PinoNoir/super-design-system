## IDE Setup

For a better developer experience, we recommend using the latest version of
Microsoft Visual Studio Code. If you don&apos;t have it installed on your device, please reach out to the IT department to assist you.

## Project Setup

All React projects at Stretto should include an `.npmrc` file at the root of the project and contain the following configuration:

```
@stretto:registry=https://nexus.stretto.com/repository/npm-releases/
registry=https://nexus.stretto.com/repository/npm-group/
strict-ssl=false
legacy-peer-deps=true
```

## CDS React Component Library Installation

To install `@stretto/cds-ui` in your project, run the following command:

```
npm install @stretto/cds-ui@latest
```

Import the component from the library&apos;s root entry to use in your project:

```jsx
// App.tsx

import { Button } from '@stretto/cds-ui';
```

## Importing the CDS Global Stylesheet

> Applications using CDS can use a set of pre-defined
> design tokens or CSS variables. Reference the [Design Tokens](https://cds.stretto.com/?path=/docs/foundations-design-tokens--overview)
> section to learn more.

1. Install `@stretto/cds-styles` in your project by running the following command:

```jsx
npm install @stretto/cds-styles
```

2. Import the minified CSS file for BCC themed applications:

```jsx
// index.css NOTE: This is the stylesheet used only for BCC themed applications

import '@stretto/cds-styles/bcc';
```

3. Import the minified CSS file for TSC themed applications:

```jsx
// index.css NOTE: This is the stylesheet used only for TSC themed applications

import '@stretto/cds-styles/tsc';
```

4. Reference the css variable in your CSS stylesheet:

```css
/* style.css */
.my-component {
  color: var(--color-neutral-100);
  background-color: var(--color-neutral-20);
}
```
