# Box

> SDS provides a custom polymorphic component.

> Note: Use the Box component as a tool for quick prototyping. If a recurring pattern is present in your application, use a specific CSS class. This will improve maintainability in the long run.

<Canvas of={BoxStories.Default} />

## Usage

- The Box component is great for prototyping and laying out content.

- By default, the box is rendered as a `div` element.

- Use the `as` prop to designate any valid html element you want to render in the DOM.

- Use the built-in CSS Prop system to style the Box component.

### Basic Inline CSS Styling

```jsx
<Box
  as="header"
  style={{
    display: 'flex',
    p: '24px',
    w: '250px',
    borderRadius: '4px',
    justifyContent: 'center',
    alignItems: 'center',
    color: 'neutral-100',
    bg: 'neutral-20',
  }}
>
  <p>This is some centered content in the Box component.</p>
</Box>
```

### CSS Prop System

```jsx
<Box as="footer" mt="24">
  <p>This is some centered content in the Box component.</p>
</Box>
```
